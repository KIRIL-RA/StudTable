# StudTable

Веб-сервис для просмотра расписания занятий

Функции:
---------
- Просмотр расписания на выбранную дату
- Отображение занятия и преподавателя 
- Автоматическая смена числителя/знаменателя

Регистрация происходит через удобный веб-интерфейс. Все аккаунты подтверждаются с аккаунта старосты.
Так же староста через свой аккаунтт вносит правки в расписание.
